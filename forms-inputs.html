Types of Inputs


Text input - takes in everything (text or String format)


Number input - takes in numbers ONLY, Also comes with Increm/Decrementer


CHECKBOX input - takes in everything


Radio Button input - circle clickable button (UNCLICKABLE, ONLY CHANGEABLE TO OTHER RADIO BUTTON)


SUBMIT input - ADDS A SUBMITTABLE BUTTON THAT CAN SUBMIT DATA TO BACKENED(use attributre value to change text shown in button)
